# Treegens Mangrove Counting API - documentation

Public developer reference for the Treegens mangrove video counting service.

- `index.html` - human-readable API reference (endpoints, async flow, result fields, client examples).
- `openapi.json` - OpenAPI 3.1 spec; point a code generator at it to produce a typed client.

Published via GitHub Pages. This repository contains documentation only; no service source code.
